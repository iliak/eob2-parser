Black Crypt items.

This list is created from items placed on dungeon and from save game (some clothes not present on dungeon).
Any item is around 20 bytes inserted every time to dungeon file, so can be partialy custom as well.

First 9 bytes means that same for all items, Axe for example:

0 01 00 8D 02 51 19 01 00 00 00 FA 00 7D 00 00 FF FF 00 00
| || || || || || || || ||
a aa bb bb cc cc de ff gg

aaa - unique number of anything placed on curent map (not level) / 12 bits
bbbb - gfx number (also determines weapon hardcoded strength or special features like Ogre Blade) / 16 bits
cccc - name offset (names present on bcdft file, starts on 115978 byte means 00 00 here) / 16 bits
d - position on sqare (Alcoves: 1 - N; 2 - E; 4 - S; 8 - W; Floor: 3 - NE; 6 - SE; 9 - NW; C - SW) / 4 bits
e - who can use item (bit flag: +1 Fighter; +2 Cleric; +4 Druid; +8 Magic User) / 4 bits
ff - item type (defines how the last bytes work) / 8 bits
gg - position in container (0-7 upper line, 8-F lower line, so Coffer uses 0, 1, 2, 8, 9, A) / 8 bits?
Some items is doubled when uses other values than normal.
X is for unique number and position on square and it's not item data itself, but must be set.
